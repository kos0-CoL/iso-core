# **App Name**: ISO CORE: Vector Lab

## Core Features:

- Procedural Vector Domain: Procedurally generates an 150x150 isometric world with biome-specific resource nodes influenced by terrain height and rivers.
- Behavioral Emergence Kernel: Multi-agent system where NPC decisions are derived from weight calculations rather than fixed scripts.
- Divine Semantic Terminal: A semantic tool powered by TensorFlow.js and Universal Sentence Encoder that applies global changes to the vector weights using natural language.
- Vector Space Inspector: Real-time manipulation panel to adjust utility weights for resources, threats, and individual agent social dynamics.
- Dynamic Resource Lifecycle: Extensive environmental system featuring consumable resources (trees, food, water) with dynamic vector modifiers based on proximity to water.
- Agent Memory Profiler: System that records agent encounters with threats and resources, affecting their risk-versus-reward utility assessments.
- Modular Execution Pipeline: Decoupled architecture ready to interface with external logic, maintaining a clear separation between render and IA simulation.

## Style Guidelines:

- The palette is inspired by mathematical visualization and research laboratories. Primary color: Scientific Digital Blue (#3B82F6).
- Background: Deep Laboratory Navy (#161920), reflecting the desaturated scientific vibe for high focus.
- Accent: Cybernetic Cyan (#0891B2), chosen via an analogous shift for a technical feel that contrasts for interaction elements.
- Headlines use 'Space Grotesk' for a high-tech feel, paired with 'Inter' for highly legible body text.
- Monospace text: 'Source Code Pro' is used for displaying all raw vector data and utility calculations.
- An asymmetric three-column dashboard optimized for observation, with specialized sidebars for global telemetry and vector editing.
- Precision transitions with slight dampening for vector changes to represent mathematical evolution rather than abrupt logic switches.
